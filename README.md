
Customer Behavior Data Analysis Project

This project represents a complete, industry standard, end-to-end data analytics workflow, designed to mirror the real responsibilities of professional analysts in modern business environments. The project encompasses all critical stages of data analysis, from data preparation and modeling to insight generation, visualization, and reporting.

Overview

This is an end-to-end data analytics project that analyzes customer shopping behaviour to uncover insights about purchasing patterns, revenue trends, and customer demographics.

The project covers the complete data analytics workflow — from raw data ingestion and cleaning in Python, to SQL-based analysis in MySQL, and finally visual storytelling through a Power BI dashboard and a presentation.

Steps Performed

Step 1 — Data Loading & Cleaning (Python)
- Loaded the CSV dataset using `pandas`
- Inspected data types, null values, and duplicates
- Cleaned column names and handled missing data
- Performed Exploratory Data Analysis (EDA)
- Exported cleaned data to MySQL using `SQLAlchemy`

Step 2 — SQL Analysis (MySQL)
Wrote and executed queries to answer business questions.

Step 3 — Power BI Dashboard
- Connected Power BI directly to MySQL database
- Created interactive visuals and KPI cards
- Built filters for gender, season, and category

Step 4 — Presentation
- Summarized key findings in a PowerPoint deck
- Included charts, insights, and business recommendations
